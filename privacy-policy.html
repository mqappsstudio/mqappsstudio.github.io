Calendar Pro — Privacy Policy

Last updated: February 7, 2026  |  Version 1.2.0

The short version: Calendar Pro does not collect, transmit, or share any of your data. Everything stays on your computer. The app works completely offline.

1. Information We DO NOT Collect

Calendar Pro does NOT collect, access, store, or transmit any of the following:

  • Your name, email address, or Microsoft account information
  • Your location, IP address, or device identifiers
  • Your browsing history or usage patterns
  • Your task descriptions, calendar entries, or any personal content
  • Your computer name, hostname, or Windows username
  • Analytics, telemetry, or crash reports to any server

2. Data Stored Locally on Your Device

Calendar Pro stores the following data only on your device, in your local app data folder:

DataLocationPurpose
Tasks & reminders%LocalAppData%\CalendarPro\tasks.jsonYour calendar data
App settings%LocalAppData%\CalendarPro\settings.jsonYour preferences
Diagnostic logs%LocalAppData%\CalendarPro\CalendarPro_Diagnostics.logTroubleshooting (see Section 3)
Task backupDocuments\CalendarPro\tasks_backup.jsonAutomatic backup of your tasks

This data is never transmitted to any server. It never leaves your device unless you explicitly export it.

3. Diagnostic Logging (New in v1.2.0)

Calendar Pro writes a diagnostic log file on your device to help troubleshoot issues. This log contains only system-level information — never personal content.

What IS logged (safe, non-personal data):
  • Windows version and build number (e.g., "Windows 11 Build 22631")
  • .NET runtime version and system architecture
  • Display scaling and total system memory
  • App version and install type (Store, sideloaded, etc.)
  • Microsoft Store license status (active/inactive)
  • Notification system status (enabled/disabled in Windows Settings)
  • Task counts (total, today's, with reminders — numbers only)
  • Error messages and code-level stack traces
  • Notification lifecycle events (sent, delivered, snoozed, failed)
What is NEVER logged:
  • Task descriptions, titles, or content of any kind
  • File paths containing your Windows username (automatically replaced with "[USER]")
  • Microsoft account email address
  • Computer name or hostname
  • IP addresses or network information
  • Any personally identifiable information

Log file management:

  • Maximum size: ~2 MB (automatically rotated — old entries are discarded)
  • You can export the log for support: Settings → Diagnostics → Export
  • You can copy the log to clipboard: Settings → Diagnostics → Copy
  • You can delete the log at any time: Settings → Diagnostics → Clear
  • The log file is stored only on your device and is never transmitted automatically

4. Network Access

Calendar Pro operates entirely offline. The app does not make any network requests. The internetClient capability in the app manifest is included only because it is required by the Windows App SDK framework — it is not used by Calendar Pro.

5. Third-Party Services

Calendar Pro does not use any third-party analytics, advertising, tracking, or crash reporting services.

6. Data Sharing

Calendar Pro does not share your data with anyone. No data leaves your device unless you explicitly export your tasks to a CSV file or export diagnostic logs — in both cases, you choose where to save the file.

7. Children's Privacy

Calendar Pro is suitable for all ages and does not collect any personal information from anyone, including children under 13.

8. Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top. Since Calendar Pro does not collect data, changes are unlikely.
